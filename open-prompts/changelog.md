# Changelog

All notable changes since the initial release are documented here.

## [1.20260302.1] - 2026-03-02

Release artifact: `OpenPrompts_1.20260302.1_x64-setup.exe`

### Added
- System tray icon support.
- New option to automatically start OpenPrompts with Windows startup.

### Fixed
- Improved `activePrompt` handling in the editor store to avoid stale/incorrect active prompt state.

### Changed
- Installer packaging updated to NSIS-only (MSI disabled).
- Version metadata bumped across app and Tauri manifests.
